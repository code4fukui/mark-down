# mark-down

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

An ex-tag forked from [marked](https://github.com/markedjs/marked).

## Demo

Checkout the [demo page](./demo/) to see mark-down in action.

## Features

- Renders Markdown to HTML
- Supports CommonMark and GitHub Flavored Markdown
- Async rendering with fetch
- Automatically updates when the `src` attribute changes

## Usage

```html
<script type="module" src="https://code4fukui.github.io/mark-down/mark-down.js"></script>
<mark-down>
# Marked in the browser

Rendered by **marked**.
</mark-down>
```
